# ðŸ’Š MedEtherial

MedEtherial is a full-stack healthcare platform built using the **MERN stack** (MongoDB, Express.js, React, Node.js). It streamlines interactions between patients and doctors by offering **secure appointment booking**, comprehensive **profile management**, and seamless access to **medical records**. All of this is delivered through a **responsive, user-friendly interface**.

## ðŸš€ Features

* **Patient & Doctor Authentication:** Secure login and registration for both user roles.
* **Appointment Booking System:** Intuitive interface for scheduling, viewing, and managing appointments.
* **Secure Medical Data Handling:** Robust measures to protect sensitive patient information.
* **Optional Admin Dashboard:** Centralized control panel for administrative tasks (if implemented).
* **Fully Responsive UI with Bootstrap:** Adapts seamlessly to various screen sizes for an optimal user experience.
* **RESTful API Integration:** Efficient communication between frontend and backend.

## ðŸ“¦ Installation & Usage

Follow these steps to get MedEtherial up and running on your local machine.

### 1. Clone the Repository

```bash
git clone [https://github.com/VJnCode/MedEtherial.git](https://github.com/VJnCode/MedEtherial.git)
cd MedEtherial
````

### 2\. Setup Backend

Navigate to the `server` directory, install dependencies, and start the development server.

```bash
cd server
npm install
npm run dev
```

#### Environment Variables

Create a `.env` file inside the `server/` folder with the following content, replacing the placeholder values:

```ini
PORT=8000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

ðŸ’¡ **Note:** Ensure your MongoDB instance is running, or connect to a MongoDB Atlas cluster.

### 3\. Setup Frontend

Navigate to the `client` directory, install dependencies, and start the development server.

```bash
cd client
npm install
npm run dev
```

### 4\. Access the Application

Once both the backend and frontend servers are running, you can access MedEtherial at:

  * **Frontend:** `http://localhost:5173`
  * **Backend API:** `http://localhost:8000`
